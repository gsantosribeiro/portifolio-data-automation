# Portfolio — Data Automation & Cleaning Scripts

Este repositório reúne projetos práticos focados em **limpeza de dados**,  
**automação**, **requisições a APIs**, e **análises rápidas** usando Python.  
É um portfólio voltado para trabalhos **freelancer** e solução de problemas reais  
de clientes que lidam com planilhas, dados bagunçados e processos manuais.

---

##  O que você encontra aqui

###  1. Scripts de Limpeza e Padronização de Planilhas (Pandas)
Ferramentas práticas que resolvem problemas comuns de clientes, como:
- Normalização de nomes de colunas  
- Remoção de acentos e caracteres inválidos  
- Conversão de tipos numéricos  
- Localização de linhas com erros  
- Detecção de valores não numéricos  
- Substituição de valores vazios por `NaN`  
- Padronização de datas e moedas  

Esses scripts são modulares e podem ser usados em **qualquer planilha**.

---

###  2. Automação com APIs
Scripts que baixam e atualizam dados automaticamente.  
Exemplos incluídos:
- Coleta de preços do Bitcoin via **CoinGecko API**
- Cálculo de médias móveis (30, 100, 150, 200 dias)
- Detecção de pontos altos/baixos recentes
- Análises simples e rápidas

Ideal para clientes que querem relatórios automáticos ou dashboards.

---

### 3. Métodos Matemáticos e Cálculo Numérico
Implementações úteis para:
- Método da bisseção  
- Propagação de erro  
- Uso de Sympy e Autograd  
- Integração numérica  
- Funções matemáticas estruturadas para engenharia e física

Mostra domínio de matemática aplicada + Python.

---

## Tecnologias usadas

- **Python 3.10+**
